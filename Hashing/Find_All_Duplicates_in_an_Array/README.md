# Find All Duplicates in an Array

## Problem

https://leetcode.com/problems/find-all-duplicates-in-an-array/

## Difficulty

Medium

## Approach

Use a hash map to count the frequency of each element. Traverse the hash map and collect all elements whose frequency is `2`.

## Time Complexity

- Time: **O(n)**

## Space Complexity

- Space: **O(n)**
