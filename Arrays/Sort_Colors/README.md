# Sort Colors

## Problem

https://leetcode.com/problems/sort-colors/

## Difficulty

Medium

## Approach

Count the occurrences of `0`, `1`, and `2` in the array. Then overwrite the array by placing all `0`s first, followed by `1`s and `2`s.

## Time Complexity

- Time: **O(n)**

## Space Complexity

- Space: **O(1)**
