# Running Sum of 1d Array

## Problem

https://leetcode.com/problems/running-sum-of-1d-array/

## Difficulty

Easy

## Approach

Traverse the array from left to right. At each index, add the previous element's running sum to the current element. This updates the array in-place to store the running sum.

## Time Complexity

- Time: **O(n)**

## Space Complexity

- Space: **O(1)**
